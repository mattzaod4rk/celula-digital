# 🧬 Célula Digital — Biologia & Informática

Jogo educativo acessível que une biologia celular e informática para estudantes do ensino médio.

## 🎮 Modos de Jogo

- **Quiz Bio+Info** — Perguntas de múltipla escolha misturando biologia e informática
- **Memória Celular** — Jogo da memória: combine organelas com suas funções
- **Código Genético** — Identifique aminoácidos a partir de códons de RNA

## ♿ Acessibilidade (WCAG 2.1 AA)

- Alto contraste e modo claro
- Leitura em voz alta (TTS)
- Fonte para dislexia
- Ajuste de tamanho de texto
- Navegação completa por teclado
- Skip link e regiões ARIA

## 🚀 Deploy

### Vercel
1. Faça push para o GitHub
2. Importe o repositório na [Vercel](https://vercel.com)
3. Deploy automático — zero configuração

### GitHub Pages
1. Vá em **Settings > Pages**
2. Selecione o branch `main` e pasta `/ (root)`
3. Salve e aguarde o deploy

### Netlify
1. Arraste a pasta do projeto para o [Netlify Drop](https://app.netlify.com/drop)
2. Ou conecte o repositório GitHub
